# Changelog

## [1.0.0] - 2024

### Adicionado
- Sistema completo de autenticação com JWT
- Plataforma para assistir múltiplas lives simultaneamente
- Criação e gerenciamento de salas de lives
- Grid responsivo para exibição de vídeos
- Integração com Redis para cache
- PostgreSQL como banco de dados
- Docker e Docker Compose para containerização
- Frontend moderno com Next.js e React
- Tailwind CSS para estilos responsivos
- React Player para reprodução de vídeos
- Zustand para gerenciamento de estado
- Toast notifications

### Features
- ✅ Autenticação segura
- ✅ Múltiplas lives simultâneas
- ✅ Criação de salas personalizadas
- ✅ Contador de espectadores
- ✅ Interface responsiva
- ✅ Cache com Redis
- ✅ API RESTful completa
